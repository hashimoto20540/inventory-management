# inventory-management

## 概要<br>
XAMPPのhtdocsディレクトリー（C:\xampp\htdocs）にinventory-managementフォルダを配置すると動作します。<br>

### 商品の在庫管理<br>
・商品の一覧を確認<br>
・商品を登録<br>
・商品を編集<br>

### ログイン機能<br>
・ユーザー登録画面<br>
・ログイン処理<br>
・ホーム画面<br>
・商品管理画面にセッションの追加
